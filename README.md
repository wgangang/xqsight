大家学习和接单的有力武器，支持代码生成。

兼容IE8及以上浏览器。支持前后端分离，使用jsonp处理跨域请求。

js文件使用闭包模块化开发，便于修改和维护

后段框架  

springmvc ＋ shiro ＋ mybatis ＋ ehcache

使用maven管理

前端框架

ace template ＋ layer ＋ selector ＋ html validate ＋ ztree

可以使用百度fis管理工具对js打包压缩处理

1.菜单管理

2.角色管理

3.字典管理

4.权限分配

5.组织机构管理

6.日志管理
